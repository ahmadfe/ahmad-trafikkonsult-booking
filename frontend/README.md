# Ahmad Trafikkonsult Driving School Booking System - Frontend

This is the frontend application built with Next.js for the driving school booking system.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Booking system for driving lessons
- User authentication
- Admin dashboard
- Mobile responsive design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- ESLint
